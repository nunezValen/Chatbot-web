---

Quiero crear una demo web para un chatbot inmobiliario de **CSFI Propiedades**.

## Tecnologías

* HTML
* CSS
* JavaScript puro
* Sin frameworks
* Sin librerías externas
* Un único archivo HTML autocontenido

## Objetivo

La web debe actuar como una interfaz de chat conectada a mi workflow de n8n.

## Endpoint

La aplicación debe realizar POST a:

```text
https://n8n-vete.onrender.com/webhook/whatsapp
```

## Formato del request

Debe enviar exactamente este JSON:

```json
{
  "body": {
    "data": {
      "from": "demo-user",
      "body": "mensaje del usuario"
    }
  }
}
```

## Persistencia de conversación

Mi workflow de n8n utiliza:

```javascript
body.data.from
```

como identificador de conversación.

Por lo tanto al cargar la página se debe generar un sessionId y almacenarlo en localStorage:

```javascript
let sessionId = localStorage.getItem('csfi-session-id');

if (!sessionId) {
  sessionId = 'demo-' + crypto.randomUUID();
  localStorage.setItem('csfi-session-id', sessionId);
}
```

Y todos los mensajes enviados deben utilizar:

```javascript
from: sessionId
```

De esta manera n8n podrá recordar la conversación.

## Formato de respuesta

El workflow devolverá:

```json
{
  "reply": "respuesta del bot"
}
```

El frontend debe leer:

```javascript
response.reply
```

y mostrarlo en el chat.

Formato:

```json
[
  {

    "text":"Busco una casa"

```

## Compatibilidad futura

Todo el chat debe estar contenido dentro de:

```html
<div id="csfi-chat-root"></div>
```

Todos los estilos deben tener prefijo:

```css
csfi-chat-
```

Ejemplos:

```css
.csfi-chat-container
.csfi-chat-header
.csfi-chat-message
.csfi-chat-user
.csfi-chat-assistant
.csfi-chat-input
.csfi-chat-send
```

No utilizar estilos globales para evitar conflictos con WordPress.
